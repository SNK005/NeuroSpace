NeuroSpace – AI-Powered Study Dashboard

NeuroSpace is a modern, AI-assisted study productivity web application designed to enhance focus, structure study sessions, and improve motivation through an interactive and minimalistic dashboard interface.

**Features**

Pomodoro-based focus timer for structured study sessions

Interactive study mode controls

Ambient sound control interface (UI-based, extendable)

Motivational quote generator for engagement and consistency

Fully responsive layout optimized for multiple devices

**Tech Stack**

HTML5

CSS3

JavaScript (Vanilla)

**AI Tools Used**

Anti-Gravity AI – Used for initial project generation and structure

ChatGPT – Used for planning, debugging, and development guidance

**Project Note**

Ambient sound functionality is currently implemented as a user interface layer and can be extended with real audio integration in future updates.
